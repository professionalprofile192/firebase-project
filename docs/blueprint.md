# **App Name**: UBL Digital Banking Login

## Core Features:

- Login Authentication: Securely authenticate users using username and password credentials.
- Forgot Password: Allow users to recover their password if forgotten.
- Corporate Enrollment: Provide a pathway for corporate users to enroll or register.

## Style Guidelines:

- Background: Light gradient from light lavender to light pink (#E6E6FA to #F4E2FF) to give a modern and welcoming feel.
- Primary: A muted purple (#737188) for buttons and active elements, providing a subtle contrast.
- Accent: White (#FFFFFF) for text and input fields, ensuring readability and clarity.
- Font: 'Inter', a sans-serif font, will be used for both body text and headings.
- The page will be split into two sections: a branding/information section on the left and the login form on the right, in a container with rounded corners for a soft, modern look.
- Input fields will have a clean, minimalist design with subtle borders or shadow.
- Subtle transition animations or micro-interactions upon successful login or when a user interacts with the input fields.
